# Data

This folder contains the data files used in the analysis scripts in this repository. The main script to generate figures is [`figures.R`](../figures.R).

Data files included here are:

| File Name | Description |
|----|----|
| [`ghg_gwp.csv`](/data/ghg_gwp.csv) | Global Warming Potential (GWP) values for various greenhouse gases |
| [`energy_densities.csv`](/data/energy_densities.csv) | Energy density values for various fuels and energy carriers |
| [`cost_table.xlsx`](/data/cost_table.xlsx) | Cost parameters for wind, solar PV, and hydrogen through 2050 |
| [`Cost_Wind_PV_H.xlsx`](/data/Cost_Wind_PV_H.xlsx) | Cost parameters for wind, solar PV, and hydrogen production technologies |
